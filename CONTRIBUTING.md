# Contributing to Holy Quran App

Thank you for your interest in improving this Quran application. We welcome contributions from everyone, but we ask that you follow these guidelines:

## Core Principles

- **Respect the sanctity of the Quran text** — The Quranic text is sacred and must be treated with utmost respect.
- **Do not modify Quranic text** without verified sources from established Islamic scholarship.
- **Maintain code quality** — Follow clean architecture principles and established patterns.
- **Test thoroughly** — All contributions must be tested before submission.

## How to Contribute

### 1. Open an Issue First

Before making major changes, please:

1. Check existing GitHub issues
2. Open a new issue describing:
   - What you want to change
   - Why you want to change it
   - How you plan to implement it

This avoids wasted effort and ensures alignment with project goals.

### 2. Development Workflow

```bash
# Fork the repository on GitHub
git clone https://github.com/YOUR-USERNAME/Holy-Quran.git
cd Holy-Quran

# Create a feature branch
git checkout -b feature/your-feature-name

# Make your changes
# Test thoroughly
flutter test

# Commit with clear messages
git commit -m "feat: description of your changes"

# Push to your fork
git push origin feature/your-feature-name

# Submit a Pull Request on GitHub
```

### 3. Pull Request Guidelines

When submitting a PR:

- **Title**: Use conventional commit format (feat:, fix:, docs:, etc.)
- **Description**: Clearly explain what and why
- **Tests**: Include tests for new features
- **Docs**: Update README if needed
- **Quran Text**: Never include unverified Quranic content

### 4. Code Standards

- Follow Dart style guide: `flutter analyze`
- Format code: `flutter format .`
- Write meaningful commit messages
- Keep commits focused and atomic

### 5. Quranic Content Guidelines

If your contribution involves Quranic text or Islamic content:

1. Use only verified sources (Tanzil, authenticated Islamic repositories)
2. Include proper attribution
3. Provide scholarly references
4. Do not alter, reorder, or misrepresent verses
5. Respect traditional Islamic typography (Uthmani Hafs)
